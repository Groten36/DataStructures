# DataStructures

A comprehensive implementation of fundamental and advanced data structures in C++ and C, demonstrating strong systems programming fundamentals and computer science principles.

## 📋 Overview

This repository showcases a well-organized collection of classic and modern data structures, built from the ground up. It's designed to demonstrate:

- **Deep understanding** of algorithmic complexity and efficiency
- **Systems programming** expertise in C and C++
- **Memory management** best practices
- **Build automation** and project organization with CMake and Makefile

## 🔧 What's Included

Explore implementations of:
- Linear Data Structures (**Arrays**, Linked Lists, **Stacks**, Queues)
- Tree Structures (Binary Trees, AVL Trees, B-Trees, etc.)
- Hash-based Data Structures (Hash Tables, Hash Sets)
- Graph Structures and Algorithms
- Heap and Priority Queues
- Advanced Structures (Tries, Segment Trees, etc.)

  The implemented structures are indicated in **bold**
